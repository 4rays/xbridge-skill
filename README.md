# xcode-mcpbridge

Claude Code plugin for integrating with Xcode via MCP mcpbridge.

## Installation

```bash
claude --plugin-dir /path/to/xcode-mcpbridge
```

## Components

- **Skills** — `skills/` directory
- **Agents** — `agents/` directory
- **Hooks** — `hooks/hooks.json`
- **MCP** — `.mcp.json` (Xcode mcpbridge server config)
